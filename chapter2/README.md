# 第2章 名前に情報を詰め込む

## 2.1 明確な単語を選ぶ
* get
* size
* stop

## 2.2 tmpやretvalなどの汎用的な名前を避ける
* retval
* tmp
* i (iterator)


## 2.3 抽象的な名前よりも具体的な名前を使う
* DISALLOW_EVIL_CONSTRUCTORS
* --run_locally


## 2.4 名前に情報を追加する
* elapsetedUrl, trustedUrl


## 2.5 名前の長さを決める
* 名前のガイドライン(長いとまずい？短いとまずい？)


## 2.6 名前のフォーマットで情報を伝える
* 定数、メンバ変数、ローカル変数
* JavaScriptのコンストラクタ関数、通常の関数、jQueryオブジェクト
